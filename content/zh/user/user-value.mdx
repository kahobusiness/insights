# 用户价值

## 定义

- 什么是用户价值：用户价值是用户的主观感受，是用户认为产品或服务能否解决问题的主观想法。我们不仅要创造逻辑上能够帮助用户解决问题的产品和服务，还要确保用户了解和认可我们产品和服务的价值。
- 经典误区
    - 产品功能不等于用户价值

## 用户价值的分类

- 实用价值：提高效率、节约时间、减少成本
- 情绪价值：提供陪伴感、增加乐趣、让人心情愉悦
- 社交价值：提供归属感或认同感

## 好产品重视用户价值

- 互联网行业产品以软件应用为主，相较于传统制造业，其产品有如下特点：
    - 产品触达成本骤降 → 产品更容易被终端用户获得 → 产品品质成为产品成功的核心要素
    - 产品可以获得用户的全方位数据 → 对用户的认知存在质的飞跃 → 个性化推荐成为可能
    - 产品生产成本骤降 → 快速迭代成为可能
- 在互联网浪潮下，产品品质对产品成功的影响，压过了用户（品牌）心智：由于触达成本更低，用户横向对比的成本也就降低了。能提供更多用户价值的产品更容易胜出
- 设计良好的互联网产品会围绕用户价值进行设计和优化，确保每个功能、界面和互动都能为用户提供明确的价值。这要求产品团队深入理解用户的需求、痛点和使用场景，并通过数据分析和用户反馈不断调整产品策略

## 用户价值不等于产品价值

- 产品价值包含两个部分，一部分是用户价值，一部分是商业价值。先有用户价值，再有商业价值，两者构成了一个产品的整体价值
- 用户价值部分，指的是产品为用户带来的用户价值增量，同时也要考虑个体价值和群体价值的关系
    - 用户价值增量 = （新价值 - 老价值）- 迁移成本
    - 用户群体价值 = 平均创造的用户价值增量 * 用户规模
- 商业价值部分，指的是产品在市场上的经济价值，包括直接收益（如销售收入、利润）和间接收益（如品牌价值提升、市场份额增加），衡量唯独则落在客单价、频次和用户量上
    - 商业价值 = 单用户价值 * 潜在用户数 =（用户愿付的价格 - 成本）* 用户频次 * 潜在用户数
