## 1. 项目概览

### 1.1 产品定位
这是一款基于 Next.js 的单人"谁是卧底"网页游戏，玩家与 AI 对手进行文字推理对抗。

### 1.2 核心玩法
- 玩家扮演玩家 1，与 AI 玩家一起参与游戏
- 每个玩家获得一个词语（平民词、卧底词或白板无词）
- 每轮游戏分为两个阶段：描述阶段 + 投票阶段
- 玩家需要通过描述和投票找出所有卧底
- 胜利条件：所有卧底被淘汰（平民胜）或 卧底数量 ≥ 平民数量（卧底胜）

### 1.3 游戏特色
- 单机游戏，无需联网
- AI 角色使用预设描述模板，模拟真实玩家
- 聊天室风格的游戏界面
- 支持自定义玩家数量、卧底数量、白板数量

## 6. 游戏规则

### 6.1 角色定义
1. **平民（Civilian）**
   - 拥有平民词语
   - 目标：找出并投票淘汰所有卧底
   - 描述策略：准确描述词语，但不能太明显

2. **卧底（Undercover）**
   - 拥有卧底词语（与平民词语相似但不同）
   - 目标：隐藏身份，混入平民
   - 描述策略：使用模糊描述，避免被发现

3. **白板（Blank）**
   - 没有词语
   - 目标：通过听取他人描述猜测词语，伪装成平民
   - 描述策略：使用通用描述，避免露馅

### 6.2 游戏流程
1. **设置阶段（Setup）**
   - 选择玩家总数（3-10 人）
   - 选择卧底数量（1 到总数一半）
   - 选择白板数量（0 到 总数-卧底数-2）

2. **查看词语阶段（View Word）**
   - 玩家查看自己的词语
   - 白板玩家看到"🤫"和"白板 - 没有词语哦！"

3. **游戏进行阶段（Playing）**
   - 每轮分为两个阶段：
     - **描述阶段（Describe）**: 所有玩家依次描述词语
     - **投票阶段（Vote）**: 所有玩家投票淘汰一名玩家

4. **游戏结束阶段（Ended）**
   - 显示所有玩家的身份和词语
   - 显示胜负结果

### 6.3 发言顺序
- **人类玩家非白板**：人类玩家先发言，然后 AI 玩家依次发言
- **人类玩家是白板**：AI 玩家先依次发言，最后人类玩家发言

### 6.4 胜利条件
- **平民胜利**：所有卧底被淘汰
- **卧底胜利**：卧底数量 ≥ 平民数量
- **玩家失败**：人类玩家被淘汰

### 6.5 重要规则
- **禁止重复描述**：任何玩家不能使用已经在本局游戏中使用过的描述（包括跨轮次和同轮次的所有描述）
- **描述验证**：人类玩家提交描述时会检查是否重复，如果重复会显示警告并拒绝提交

## 10. 游戏流程

### 10.1 开始游戏（startGame）

**功能**：初始化游戏，分配角色和词语

**步骤**：
1. 随机选择一组词语对
2. 创建角色数组：
   - 添加 `undercoverCount` 个卧底角色（卧底词）
   - 添加 `blankCount` 个白板角色（"白板"）
   - 添加剩余数量的平民角色（平民词）
3. 打乱角色数组（Fisher-Yates 洗牌算法）
4. 创建玩家数组：
   - 玩家 1（索引 0）设为人类玩家
   - 其余为 AI 玩家
5. 切换到 `viewWord` 状态

**代码逻辑**：
```typescript
const startGame = () => {
  const wordPair = wordPairs[Math.floor(Math.random() * wordPairs.length)];
  const playerRoles: Array<{ role: PlayerRole; word: string }> = [];

  // 添加角色
  for (let i = 0; i < undercoverCount; i++) {
    playerRoles.push({ role: 'undercover' as PlayerRole, word: wordPair.undercover });
  }
  for (let i = 0; i < blankCount; i++) {
    playerRoles.push({ role: 'blank' as PlayerRole, word: '白板' });
  }
  for (let i = 0; i < totalPlayers - undercoverCount - blankCount; i++) {
    playerRoles.push({ role: 'civilian' as PlayerRole, word: wordPair.civilian });
  }

  // 洗牌
  for (let i = playerRoles.length - 1; i > 0; i--) {
    const j = Math.floor(Math.random() * (i + 1));
    [playerRoles[i], playerRoles[j]] = [playerRoles[j], playerRoles[i]];
  }

  // 创建玩家
  const newPlayers = playerRoles.map((p, i) => ({
    id: i + 1,
    ...p,
    alive: true,
    isHuman: i === 0
  }));

  setPlayers(newPlayers);
  setHumanPlayer(newPlayers[0]);
  setGameState('viewWord');
  setShowWord(false);
  setMessages([]);
  setUsedDescriptions({});
};
```

```typescript

const systemMsg: Message = {
  type: 'system' as MessageType,
  content: `第 ${round} 轮开始！`,
  timestamp: new Date()
};
setMessages([systemMsg]);

const aiPlayers = players.filter(p => p.alive && !p.isHuman);
const roundDescriptions: string[] = [];

aiPlayers.forEach((player, index) => {
  setTimeout(() => {
    const aiDesc = generateDescription(player.word, player.role, player.id, usedDescriptions, roundDescriptions);
    roundDescriptions.push(aiDesc);

    setUsedDescriptions(prev => ({
      ...prev,

    setMessages(prev => [...prev, {
      type: 'ai' as MessageType,
      playerId: player.id,

    // 最后一个 AI 发言后提示人类玩家

      setTimeout(() => {
        setMessages(prev => [...prev, {
          type: 'system' as MessageType,
          content: '💬 轮到你了，请描述你的词语',
          timestamp: new Date()
        } as Message]);
      }, 500);
    }
  }, index * 800);
});
```

### 10.3 提交描述（submitDescription）

**功能**：人类玩家提交描述

**步骤**：
1. **验证描述不为空**
2. **检查描述是否重复**：
   - 检查所有玩家的历史描述（`usedDescriptions`）
   - 检查当前轮次的消息（`messages`）
   - 如果重复，显示警告并拒绝提交
3. **记录人类玩家的描述**到 `usedDescriptions`
4. **添加用户消息**到消息列表
5. **触发 AI 发言**或**进入投票阶段**：

#### 情况 1：人类玩家是白板（已是最后发言）
```typescript
setTimeout(() => {

    type: 'system' as MessageType,
    content: '所有玩家已完成描述，现在开始投票！',
    timestamp: new Date()
  } as Message]);
  setCurrentPhase('vote');
}, 1000);
```

```typescript
// 所有 AI 发言完毕后进入投票

    type: 'system' as MessageType,
    content: '所有玩家已完成描述，现在开始投票！',

  } as Message]);
  setCurrentPhase('vote');
}, (aiPlayers.length + 1) * 800 + 500);
```

### 10.4 提交投票（submitVote）

**功能**：人类玩家投票并触发 AI 投票

**步骤**：
1. 验证已选择投票目标
2. 创建投票结果对象，记录人类玩家的投票
3. 添加人类玩家的投票消息
4. **立即切换到描述阶段**（防止重复提交）
5. 收集本轮所有玩家的描述（用于 AI 投票决策）
6. AI 玩家依次投票（每个间隔 600ms）
7. 延迟后显示投票结果

```typescript
const submitVote = () => {
  if (selectedVote === null) return;

  const votes: VoteResults = { [selectedVote]: 1 };

  // 添加人类玩家的投票消息
  setMessages(prev => [...prev, {
    type: 'vote' as MessageType,
    playerId: 1,
    playerName: '你',
    content: `投票给 ${getAIName(selectedVote)}`,
    votedFor: selectedVote,
    timestamp: new Date()
  } as Message]);

  // 切换到描述阶段（防止重复提交）
  setCurrentPhase('describe');
  setSelectedVote(null);

  // 收集描述
  const aiDescriptions: Descriptions = {};
  messages.forEach(msg => {
    if ((msg.type === 'user' || msg.type === 'ai') && msg.playerId) {
      aiDescriptions[msg.playerId] = msg.content;
    }
  });

  // AI 依次投票
  players.forEach(player => {
    if (player.alive && !player.isHuman) {
      const vote = generateAIVote(player, players, aiDescriptions);
      votes[vote] = (votes[vote] || 0) + 1;

      setTimeout(() => {
        setMessages(prev => [...prev, {
          type: 'vote' as MessageType,
          playerId: player.id,
          playerName: getAIName(player.id),
          content: `投票给 ${vote === 1 ? '你' : getAIName(vote)}`,
          votedFor: vote,
          timestamp: new Date()
        } as Message]);
      }, (player.id - 1) * 600);
    }
  });

  setVoteResults(votes);

  // 显示投票结果
  setTimeout(() => {
    showVoteResultsAndExecute(votes);
  }, players.length * 600 + 500);
};
```

### 10.5 显示投票结果并执行淘汰（showVoteResultsAndExecute）

**功能**：计算投票结果，淘汰玩家，检查游戏结束条件

**步骤**：
1. 计算得票最多的玩家
2. 显示淘汰结果消息
3. 更新玩家状态（设置 `alive: false`）
4. 检查游戏结束条件：
   - 人类玩家被淘汰 → 游戏结束
   - 所有卧底被淘汰 → 游戏结束
   - 卧底数量 ≥ 平民数量 → 游戏结束
5. 如果游戏未结束：
   - 增加轮次
   - 重置状态
   - 显示新一轮开始消息

```typescript
const showVoteResultsAndExecute = (votes: VoteResults) => {
  let maxVotes = 0;
  let eliminatedId: number | null = null;

  Object.entries(votes).forEach(([id, count]) => {
    if (count > maxVotes) {
      maxVotes = count;
      eliminatedId = parseInt(id);
    }
  });

  if (eliminatedId === null) return;

  const resultText = `投票结果：${eliminatedId === 1 ? '你' : getAIName(eliminatedId)} 被淘汰（${maxVotes} 票）`;

  setMessages(prev => [...prev, {
    type: 'result' as MessageType,
    content: resultText,
    eliminatedId: eliminatedId,
    timestamp: new Date()
  } as Message]);

  const newPlayers = players.map(p =>
    p.id === eliminatedId ? { ...p, alive: false } : p
  );

  setPlayers(newPlayers);

  const aliveRoles = newPlayers.filter(p => p.alive);
  const aliveUndercovers = aliveRoles.filter(p => p.role === 'undercover').length;
  const aliveCivilians = aliveRoles.filter(p => p.role === 'civilian').length;
  const humanAlive = newPlayers.find(p => p.isHuman)?.alive;

  setTimeout(() => {
    // 检查游戏结束条件
    if (!humanAlive || aliveUndercovers === 0) {
      setGameState('ended');
      return;
    }

    if (aliveUndercovers >= aliveCivilians) {
      setGameState('ended');
      return;
    }

    // 继续下一轮
    setRound(round + 1);
    setSelectedVote(null);
    setVoteResults({});
    setCurrentPhase('describe');

    setMessages(prev => [...prev, {
      type: 'system' as MessageType,
      content: `第 ${round + 1} 轮开始！继续描述词语`,
      timestamp: new Date()
    } as Message]);
  }, 2000);
};
```

## 11. AI 逻辑

### 11.1 AI 描述生成（generateDescription）

**功能**：为 AI 玩家生成描述

**参数**：
- `word`: 玩家的词语
- `role`: 玩家角色
- `playerId`: 玩家 ID
- `usedDescs`: 所有玩家历史使用的描述
- `currentRoundUsed`: 当前轮次已使用的描述

**逻辑**：
1. 获取所有历史使用过的描述（跨轮次）
2. 根据角色选择描述池：
   - **白板**：使用 `blankDescriptions`
   - **平民/卧底**：使用 `descriptionTemplates[word]`
3. 过滤掉已使用的描述（历史 + 当前轮次）
4. 如果没有可用描述，随机选择一个
5. 返回描述

```typescript
const generateDescription = (
  word: string,
  role: PlayerRole,
  playerId: number,
  usedDescs: Record<number, string[]>,
  currentRoundUsed: string[] = []
): string => {
  const allHistoricalDescs = Object.values(usedDescs).flat();

  if (role === 'blank') {
    const availableBlank = blankDescriptions.filter(
      desc => !allHistoricalDescs.includes(desc) && !currentRoundUsed.includes(desc)
    );
    if (availableBlank.length === 0) {
      return blankDescriptions[Math.floor(Math.random() * blankDescriptions.length)];
    }
    return availableBlank[Math.floor(Math.random() * availableBlank.length)];
  }

  const templates = descriptionTemplates[word as keyof typeof descriptionTemplates];
  if (!templates) return '这个东西很常见';

  const availableDescs = templates.filter(
    desc => !allHistoricalDescs.includes(desc) && !currentRoundUsed.includes(desc)
  );

  if (availableDescs.length === 0) {
    return templates[Math.floor(Math.random() * templates.length)];
  }

  return availableDescs[Math.floor(Math.random() * availableDescs.length)];
};
```

### 11.2 AI 投票逻辑（generateAIVote）

**功能**：AI 玩家根据描述决定投票目标

**参数**：
- `player`: 当前投票的 AI 玩家
- `allPlayers`: 所有玩家数组
- `allDescriptions`: 所有玩家的描述

**评分机制**：
1. **基础分**：随机 0-30 分
2. **模糊描述加分**（+40 分）：
   - 描述长度 < 15
   - 包含 "..."、"挺"、"这个"、"不好说"、"难形容"
3. **词语矛盾加分**（+35 分）：
   - 平民玩家的特定词语检测：
     - 面条玩家：描述包含"米"或"云南"
     - 米线玩家：描述包含"面"或"小麦"
     - 西瓜玩家：描述包含"冬"、"汤"或"蔬菜"
     - 冬瓜玩家：描述包含"夏"、"甜"或"汁"
4. **卧底策略加分**（+25 分）：
   - 如果当前玩家是卧底，优先投票给平民

**投票决策**：
- 70% 概率投票给可疑度最高的玩家
- 30% 概率投票给可疑度第二高的玩家

```typescript
const generateAIVote = (player: Player, allPlayers: Player[], allDescriptions: Descriptions): number => {
  const alivePlayers = allPlayers.filter((p: Player) => p.alive && p.id !== player.id);

  const suspiciousness = alivePlayers.map((p: Player) => {
    let score = Math.random() * 30;

    const desc = allDescriptions[p.id] || '';

    // 模糊描述加分
    if (desc.length < 15 ||
        desc.includes('...') ||
        desc.includes('挺') ||
        desc.includes('这个') ||
        desc.includes('不好说') ||
        desc.includes('难形容')) {
      score += 40;
    }

    // 词语矛盾加分
    if (player.role === 'civilian') {
      if (player.word === '面条' && (desc.includes('米') || desc.includes('云南'))) score += 35;
      if (player.word === '米线' && (desc.includes('面') || desc.includes('小麦'))) score += 35;
      if (player.word === '西瓜' && (desc.includes('冬') || desc.includes('汤') || desc.includes('蔬菜'))) score += 35;
      if (player.word === '冬瓜' && (desc.includes('夏') || desc.includes('甜') || desc.includes('汁'))) score += 35;
    }

    // 卧底策略加分
    if (player.role === 'undercover' && p.role === 'civilian') {
      score += 25;
    }

    return { player: p, score };
  });

  suspiciousness.sort((a, b) => b.score - a.score);

  if (Math.random() < 0.7 || suspiciousness.length === 1) {
    return suspiciousness[0].player.id;
  } else {
    return suspiciousness[1].player.id;
  }
};
```
